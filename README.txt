Código desenvolvido por Mateus de Freitas Bueno, Brendon Vinicius Kulak e Leonardo Bellincanta de Souza.

Segue a descrição de cada arquivo .m:
- varrendo_fs_d.m : plota o diagrama de bode de todos os modelos frente a variações da frequencia de chaveamento e da razão cíclica. As arquivos (pdf) exportados vão para pasta figures\varrendo_fs_d.
- valida_begin_time.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para dente de serra decrescente.
- valida_end_time.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para dente de serra crescente.
- valida_sym_off.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para o triangular com sincronização no pico.
- valida_sym_on.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para o triangular com sincronização no vale.
- valida_double.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para a dupla atualização.
- valida_double.m :plota o diagrama de bode da simulação do psim junto com o modelo matematico, isso para a dupla atualização.

Descrição das pastas:
- Moduladores_Digitais\functions :onde se encotra todas as funções desenvolvidas para esse trabalho.
- Moduladores_Digitais\simulacoes :onde se encontra todas as simulações no Psim e os dados dos diagramas de bode simulados.
- Moduladores_Digitais\figures :onde são salvas todas as figuras plotadas.
